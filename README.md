## Criterias

### Posting Users
-   [ ] Email is unique;
-   [ ] Email is mandatory.

### Posting Tags
-   [ ] Tag name is unique;
-   [ ] Tag name is mandatory;
-   [ ] Only administrators can post tags.

### Posting Compliments
-   [ ] The sender of compliment can't be the receiver;
-   [ ] Only auth user can post a compliment.
